- Name: tfsites.VisualizeTFSitesOnSequences
- Version: Updated docker container to v17.0.19.
- Language: any
- Description: Annotates transcription factor binding sites across a DNA sequence with optional motif database generation. Multiple transcription factors can be analyzed. Each binding site is labeled with a unique binding site ID and its start and end position. If reference data is provided for a transcription factor, the affinity/score of this site will be labeled and the intensity of the binding site's color will be proportional. At least one of affinity information and files or batch motif files (or both) must be provided. If motif database parameters are provided, it will first generate a motif database and use it for visualization.
